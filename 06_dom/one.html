We can have access to many method in document as console.log(document.baseURI) console.log(document.links) etc

console.log(document.links) which give me HTMLCollection where HTMLCollection, NodeCollection and Array they are not array they are converted to array. as array property is not available on them

console.log(document.links[0].href); // gives the href of the first link

console.log(document.links.length); // gives the number of links in the document

console.log(document.links.item(0)); // gives the first link element

console.log(document.links.namedItem('linkName')); // gives the link with name 'linkName'

We can also use forEach on HTMLCollection after converting it to array

document.querySelector('.bg-black') // selects the first element with class 'bg-black'

document.querySelectorAll('.bg-black') // selects all elements with class 'bg-black'

we can manipulate the DOM using methods like below:

document.createElement('div') // creates a new div element

document.createTextNode('Hello World') // creates a new text node with 'Hello World'

document.getElementById('elementId') // selects the element with id 'elementId'

document.getElementsByClassName('className') // selects all elements with class 'className'

document.getElementsByTagName('tagName') // selects all elements with tag name 'tagName'

document.querySelector('.bg-black').style.backgroundColor = 'blue'; // changes the background color of the first element with class 'bg-black' to blue

document.querySelector('.bg-black').style.color = 'white'; // changes the text color of the first element with class 'bg-black' to white

we can change the text of DOM as below:

document.querySelector('.heading').textContent = 'New Heading'; // changes the text of the first element with class 'heading' to 'New Heading'
